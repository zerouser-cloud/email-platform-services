    **2. Create `apps/auth/src/infrastructure/persistence/schema/users.schema.ts` (per D-01, D-02, D-04, D-05):**
    - Import `pgSchema, uuid, varchar, timestamp` from `drizzle-orm/pg-core`.
    - Export `authSchema = pgSchema('auth')`.
    - Export `users = authSchema.table('users', { ... })` with columns:
      - `id: uuid('id').primaryKey()` -- no `.defaultRandom()`, app generates UUID (D-04).
      - `email: varchar('email', { length: 255 }).notNull().unique()` -- unique constraint.
      - `passwordHash: varchar('password_hash', { length: 255 }).notNull()` -- camelCase key maps to snake_case DB column.
      - `role: varchar('role', { length: 50 }).notNull()`
      - `organization: varchar('organization', { length: 255 }).notNull()`
      - `team: varchar('team', { length: 255 }).notNull()`
      - `createdAt: timestamp('created_at').defaultNow().notNull()` -- DB default (D-05).
      - `updatedAt: timestamp('updated_at').defaultNow().notNull()` -- DB default (D-05).

    **3. Create `apps/auth/src/infrastructure/persistence/schema/index.ts`:**
    - Barrel: `export { authSchema, users } from './users.schema';`

    **4. Create `apps/auth/src/infrastructure/persistence/user.mapper.ts` (per D-03, D-10, D-11):**
    - Import `User` from `../../domain/entities/user.entity`.
    - Import `type { users }` from `./schema/users.schema`.
    - Define `type UserRow = typeof users.$inferSelect` and `type NewUserRow = typeof users.$inferInsert` -- these types stay in THIS file only (D-11).
    - Export `UserMapper` object with:
      - `toDomain(row: UserRow): User` -- maps only 5 business fields (id, email, role, organization, team). Skips passwordHash, createdAt, updatedAt (D-03, D-05).
      - `toPersistence(user: User, passwordHash: string): NewUserRow` -- takes passwordHash as separate arg since domain entity does not carry it (D-03). Omits createdAt/updatedAt (DB defaults).

    **5. Create `apps/auth/src/infrastructure/persistence/pg-user.repository.ts` (per D-12, D-13):**
    - Import `Inject, Injectable` from `@nestjs/common`.
    - Import `eq` from `drizzle-orm` (NOT from `drizzle-orm/pg-core`).
    - Import `type { NodePgDatabase }` from `drizzle-orm/node-postgres` -- NO schema generic (D-13, research Pattern 6).
    - Import `DRIZZLE` from `@email-platform/foundation`.
    - Import `type { UserRepositoryPort }` from `../../application/ports/outbound/user-repository.port`.
    - Import `type { User }` from `../../domain/entities/user.entity`.
    - Import `users` from `./schema/users.schema`.
    - Import `UserMapper` from `./user.mapper`.
    - `@Injectable() export class PgUserRepository implements UserRepositoryPort`:
      - Constructor: `@Inject(DRIZZLE) private readonly db: NodePgDatabase`.
      - `async findByEmail(email: string): Promise<User | null>` -- `db.select().from(users).where(eq(users.email, email)).limit(1)`, return `UserMapper.toDomain(row)` or `null`.
      - `async save(user: User): Promise<void>` -- `db.insert(users).values(UserMapper.toPersistence(user, '')).onConflictDoUpdate({ target: users.id, set: { email, role, organization, team, updatedAt: new Date() } })`. Empty string for passwordHash is intentional stub (real password handling deferred).

    **6. Create `apps/auth/drizzle.config.ts` (per D-06, D-07, D-09):**
    - Import `defineConfig` from `drizzle-kit`.
    - Export default `defineConfig({ dialect: 'postgresql', schema: './src/infrastructure/persistence/schema/index.ts', out: './drizzle', dbCredentials: { url: process.env.DATABASE_URL! }, schemaFilter: ['auth'], migrations: { table: '__drizzle_migrations', schema: 'auth' } })`.
    - NOTE: This is a CLI config file -- `process.env.DATABASE_URL!` is acceptable here (not runtime app code).

    **7. Add scripts to `apps/auth/package.json`:**
    - Add `"db:generate": "drizzle-kit generate"` and `"db:migrate": "drizzle-kit migrate"` to scripts.

    **Anti-patterns to avoid:**
    - Do NOT import anything from `drizzle-orm` in domain/ or application/ directories.
    - Do NOT use `NodePgDatabase<typeof schema>` generic -- plain `NodePgDatabase` is sufficient for `select().from()` API.
    - Do NOT add `.defaultRandom()` to the `id` column -- app generates UUIDs.
    - Do NOT put `drizzle.config.ts` inside `src/` -- it belongs at `apps/auth/` root.
