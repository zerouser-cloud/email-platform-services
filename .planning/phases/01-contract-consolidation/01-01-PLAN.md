  <action>
    **Per D-06 — Add `generate` task to turbo.json:**
    Add a new `"generate"` task to the `tasks` object in `turbo.json` with this exact configuration:
    ```json
    "generate": {
      "inputs": ["proto/**/*.proto", "scripts/generate.sh"],
      "outputs": ["src/generated/**"]
    }
    ```
    - `inputs`: Only proto source files and the generation script affect the cache hash. Changes to other files will NOT invalidate the generate cache.
    - `outputs`: Turbo caches `src/generated/**` for restoration on cache hits.
    - No `dependsOn` needed — generate has no upstream dependencies.
    - This task is defined globally but only `@email-platform/contracts` has a `generate` script in its package.json, so Turbo silently skips it for all other packages.

    **Per D-07 — Update `build` task dependsOn:**
    Change the `build` task's `dependsOn` from `["^build"]` to `["^build", "generate"]`.
    - `"^build"` (with caret) = run build in upstream dependency packages first.
    - `"generate"` (NO caret) = run generate in the SAME package before build. This is critical — using `"^generate"` would be wrong (it means upstream packages, not same package).
    - For packages without a `generate` script, Turbo silently skips the generate dependency.

    **Per CONT-03 — Rename root script:**
    In root `package.json`, rename the script key from `"proto:generate"` to `"generate:contracts"`. The value stays the same: `"pnpm --filter @email-platform/contracts run generate"`.

    The final turbo.json should look like:
    ```json
    {
      "$schema": "https://turbo.build/schema.json",
      "globalEnv": ["NODE_ENV"],
      "tasks": {
        "generate": {
          "inputs": ["proto/**/*.proto", "scripts/generate.sh"],
          "outputs": ["src/generated/**"]
        },
        "build": {
          "dependsOn": ["^build", "generate"],
          "outputs": ["dist/**"]
        },
        "dev": {
          "cache": false,
          "persistent": true,
          "passThroughEnv": ["*"]
        },
        "lint": {
          "dependsOn": ["^lint"]
        },
        "typecheck": {
          "dependsOn": ["^build"]
        },
        "start": {
          "dependsOn": ["^build"],
          "cache": false,
          "persistent": true,
          "passThroughEnv": ["*"]
        }
      }
    }
    ```
  </action>
