  <action>
    Create the project structure and all infrastructure resources in Coolify UI.

    **Step A: Create Project and Environments (per D-01)**
    1. Go to **Projects** -> **+ Add**
    2. Name: `Email Platform`
    3. Inside the project, create two environments: `dev` and `production`

    **Step B: Create Infrastructure in DEV environment (per D-02, D-13-D-16)**

    Switch to `dev` environment, then create these resources:

    1. **PostgreSQL** (per D-13):
       - **+ Add Resource** -> **Database** -> **PostgreSQL**
       - Version: 16
       - Note the assigned hostname (e.g., `postgres-<uuid>`) -- you'll need it for DATABASE_URL
       - Note the generated password

    2. **Redis** (per D-14):
       - **+ Add Resource** -> **Database** -> **Redis**
       - Version: 7
       - Note the assigned hostname for REDIS_URL

    3. **RabbitMQ** (per D-15):
       - **+ Add Resource** -> **Service** -> search "RabbitMQ"
       - Note the assigned hostname for RABBITMQ_URL

    4. **Garage** (per D-16):
       - **+ Add Resource** -> **Service** -> search "Garage"
       - This is S3-compatible storage replacing MinIO on the server
       - Note the hostname, S3 API port (default 3900)
       - After Garage starts, you'll need to create a bucket and access key (documented in Plan 03)

    **Step C: Create Infrastructure in PRODUCTION environment**

    Switch to `production` environment and repeat Step B (separate instances per D-13-D-16):
    - PostgreSQL 16 (note hostname, password)
    - Redis 7 (note hostname)
    - RabbitMQ (note hostname)
    - Garage (note hostname)

    **Record all hostnames and credentials** -- you'll need them for Plan 03 env var configuration.

    ```
    - Garage hostname: _______________

    PRODUCTION:
    - PostgreSQL hostname: _______________  password: _______________
    - Redis hostname: _______________
    - RabbitMQ hostname: _______________
    - Garage hostname: _______________
    ```
