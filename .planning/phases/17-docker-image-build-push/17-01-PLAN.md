  <action>
Create `.github/workflows/docker-build.yml` with the following exact structure (per D-01 through D-14):

**Workflow trigger (per D-01, D-02):**
- `name: Docker Build & Push`
- `on: push` to branches `[dev, main]`

**Concurrency (per D-03):**
- `group: docker-${{ github.ref }}`
- `cancel-in-progress: true`

**Job: `build-and-push`** (per D-09):
- `name: Build ${{ matrix.service }}`
- `runs-on: ubuntu-latest`
- `permissions: { contents: read, packages: write }` (explicit packages:write for GHCR push)
- `strategy.matrix.service: [gateway, auth, sender, parser, audience, notifier]`

**Steps:**
1. `actions/checkout@v4` — no special fetch-depth needed (not using git history)
2. `docker/setup-buildx-action@v4` — sets up Buildx with GHA cache v2 support
3. `docker/login-action@v4` — with `registry: ghcr.io`, `username: ${{ github.actor }}`, `password: ${{ secrets.GITHUB_TOKEN }}` (per D-13, D-14)
4. `docker/metadata-action@v6` with id `meta` (per D-04, D-05, D-07, D-08):
   - `images: ghcr.io/${{ github.repository_owner }}/email-platform-${{ matrix.service }}`
   - Tags block with 4 rules:
     - `type=sha,prefix=dev-,enable=${{ github.ref == 'refs/heads/dev' }}` — produces `dev-abc1234` on dev
     - `type=raw,value=dev-latest,enable=${{ github.ref == 'refs/heads/dev' }}` — produces `dev-latest` on dev
     - `type=sha,prefix=,enable=${{ github.ref == 'refs/heads/main' }}` — produces `abc1234` on main
     - `type=raw,value=latest,enable=${{ github.ref == 'refs/heads/main' }}` — produces `latest` on main
5. `docker/build-push-action@v7` (per D-10, D-11, D-12):
   - `context: .`
   - `file: infra/docker/app.Dockerfile`
   - `push: true`
   - `tags: ${{ steps.meta.outputs.tags }}`
   - `labels: ${{ steps.meta.outputs.labels }}`
   - `build-args: APP_NAME=${{ matrix.service }}`
   - `cache-from: type=gha,scope=${{ matrix.service }}`
   - `cache-to: type=gha,mode=max,scope=${{ matrix.service }}`

Do NOT add `platforms:` — rely on runner default (linux/amd64). Do NOT modify the existing Dockerfile or ci.yml. No semver tags (per D-06).
