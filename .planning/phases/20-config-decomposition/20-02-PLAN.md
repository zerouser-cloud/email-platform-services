Migrate all 6 services from `loadGlobalConfig()` to `loadConfig(XxxEnvSchema)` in main.ts and from `AppConfigModule` to `AppConfigModule.forRoot(XxxEnvSchema)` in root module.

**Pattern for each main.ts** (example: auth):

Before:
```typescript
import { SERVICE, loadGlobalConfig } from '@email-platform/config';
// ...
const config = loadGlobalConfig();
```

After:
```typescript
import { SERVICE, loadConfig } from '@email-platform/config';
import { AuthEnvSchema } from './infrastructure/config';
// ...
const config = loadConfig(AuthEnvSchema);
```

**Pattern for each root module** (example: auth):

Before:
```typescript
import { AppConfigModule } from '@email-platform/config';
// ...
imports: [AppConfigModule, ...]
```

After:
```typescript
import { AppConfigModule } from '@email-platform/config';
import { AuthEnvSchema } from './infrastructure/config';
// ...
imports: [AppConfigModule.forRoot(AuthEnvSchema), ...]
```
